== Description ==

A plugin for wordpress to select which custom type on "Recent Posts" widget. 
Inspired to: Enhanced Recent Posts (http://enhanced-recent-posts.vincentprat.info)

== Installation ==

1. Upload `custom-type-recent-posts.zip` and expand it to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
