The Plugin Template provides a convenient starting place for WordPress plugin development.

== Description ==

This is simply a template for plugin creation. It eleminates the need to repeat the baby steps every time a new plugin is started. This readme file is in the proper format for plugins listed in the WordPress.org repository.


== Installation ==

1. Upload the `plugin-template` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Change the Plugin Template options via the Settings > Plugin Template page of the WordPress administration area.
